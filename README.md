# LCA_C1_Core_Project_ModernTech_HR_System

## Overview

The ModernTech HR Management System is a Human Resources web application built using Vue.js 3 and Vite. The application helps manage employees, payroll, leave requests, and attendance, and provides a dashboard that summarises important HR information.

The project demonstrates the use of Vue components, props, events, data binding, conditional rendering, and localStorage to build a responsive single-page application.

## Technologies Used

- Vue.js 3
- Vite
- JavaScript
- HTML
- Bootstrap 5

## Features

### Dashboard
- Displays total employees
- Displays active employees
- Displays pending leave requests
- Displays total monthly payroll
- Displays attendance statistics

### Employee Management
- View employee records
- Add new employees
- Edit employees
- Delete employees
- Automatically updates dashboard and payroll information

### Payroll Management
- Calculate hourly rates and hours worked
- Calculate tax deductions (15%)
- Calculate net salary
- Generate digital payslips for employees

### Leave Management
- Submit leave requests
- Approve leave requests
- Deny leave requests
- Automatically updates pending leave statistics

### Attendance Tracking
- Mark employees as Present or Absent
- View attendance summary
- Dashboard updates attendance statistics automatically

### Data Persistence
Employee and leave request data are stored using localStorage. Information remains available after refreshing the browser.

## Installation

1. Clone the repository:
```
https://github.com/JosephBanda135/LCA_C1_Core_Project_ModernTech_HR_System.git
```

2. Navigate into the project folder:
```
cd moderntech-hr-system
```

3. Install dependencies:
```
npm install
```

4. Start the development server:
```
npm run dev
```

5. Open the local URL displayed in the terminal.
